# Checklist

## 功能验证

### 知识内容管理
- [ ] 系统支持上传PDF文档并正确解析文本内容
- [ ] 系统支持上传Word文档并正确解析文本内容
- [ ] 系统支持上传Excel文件并正确解析内容
- [ ] 系统支持上传Markdown文件并正确解析内容
- [ ] 系统支持手动创建知识条目（标题、内容、分类、标签）
- [ ] 系统支持编辑已有知识条目
- [ ] 系统支持删除知识条目
- [ ] 系统支持查看知识条目详情

### 知识组织体系
- [ ] 系统支持树形目录结构展示（制造运营/仓储物流/质量管理/设备管理）
- [ ] 系统支持按目录层级浏览知识内容
- [ ] 系统支持创建和管理多维标签
- [ ] 系统支持为知识条目添加多个标签
- [ ] 系统支持按标签筛选知识条目
- [ ] 系统在知识详情页展示同分类/同标签的相关内容

### Skill管理系统（核心）
- [ ] 系统提供Skill管理页面
- [ ] 系统支持创建新Skill（名称、描述、分类、功能、触发条件、Prompt模板、知识范围）
- [ ] 系统支持查看Skill列表
- [ ] 系统支持编辑已有Skill
- [ ] 系统支持删除Skill
- [ ] 系统支持按模块维度（制造运营/仓储物流/质量管理/设备管理）分类查看Skill
- [ ] 系统支持按功能维度分类查看Skill
- [ ] 系统支持启用/禁用Skill
- [ ] 系统支持为Skill关联知识范围（分类/标签）
- [ ] 系统支持Prompt模板编辑（含变量占位符）
- [ ] 系统后端提供Skill路由测试接口

### Skill路由引擎
- [ ] 系统根据用户问题匹配对应Skill（关键词匹配）
- [ ] 系统根据用户问题匹配对应Skill（语义匹配）
- [ ] 系统在Skill关联的知识范围内检索
- [ ] 系统使用Skill的Prompt模板生成答案
- [ ] 无匹配Skill时，系统使用默认Skill在全量知识库检索

### 智能搜索
- [ ] 系统支持语义搜索，返回语义相关的结果
- [ ] 系统支持关键词搜索（SQLite LIKE查询）
- [ ] 搜索结果按相关度排序
- [ ] 搜索结果高亮显示匹配内容

### 智能问答
- [ ] 系统支持基于知识库的智能问答
- [ ] 系统通过Skill路由精准检索知识
- [ ] 系统生成的答案标注来源出处
- [ ] 系统保存问答历史记录
- [ ] 系统提供问答历史页面（列表查看、搜索、删除）
- [ ] 无匹配Skill时使用默认Skill回答

### 系统首页/仪表盘
- [ ] 系统提供首页，展示统计卡片（知识总数、Skill数、今日问答数、文档数）
- [ ] 系统首页展示最近问答记录

### 知识内容编辑器
- [ ] 系统提供Markdown编辑器用于知识内容编辑
- [ ] 编辑器支持工具栏（标题、列表、代码块、加粗等）
- [ ] 编辑器支持实时预览

### 向量索引自动同步
- [ ] 新增知识时自动生成向量索引
- [ ] 更新知识时自动重建向量索引
- [ ] 删除知识时自动移除向量索引
- [ ] 数据库与向量数据库保持一致

### 搜索与分页
- [ ] 搜索结果支持分页
- [ ] 知识列表支持分页
- [ ] 问答历史支持分页

### API文档与健康检查
- [ ] 系统提供Swagger API文档（/docs）
- [ ] 系统提供健康检查端点（/health）
- [ ] 健康检查返回数据库、向量数据库、模型配置状态

### 相关内容推荐
- [ ] 系统在知识详情页展示相关推荐
- [ ] 推荐基于同分类或同标签
- [ ] 推荐结果可点击跳转

### 交互体验优化
- [ ] 系统支持批量多文件上传（拖拽+队列管理+进度显示）
- [ ] 系统支持上传后批量编辑元数据（统一设置分类标签）

- [ ] 系统支持知识批量操作（删除/打标签/移动分类）
- [ ] 系统支持答案反馈（点赞/点踩）
- [ ] 系统支持快捷追问推荐
- [ ] 系统支持Skill模板创建
- [ ] 系统记录搜索历史，支持快捷重复搜索
- [ ] 系统首次启动无模型配置时引导用户配置
- [ ] 系统提供面包屑导航

### Token统计
- [ ] 系统记录每次AI调用的token消耗（输入/输出token）

- [ ] 系统支持按时间/Skill/模型维度查看统计

### Token优化
- [ ] 系统通过Skill路由限定检索范围，减少上下文token

- [ ] 相似问题命中缓存时直接返回结果
- [ ] Token消耗相比直接调用LLM显著降低

### 模型配置管理（核心）
- [ ] 系统提供模型配置管理页面
- [ ] 系统支持查看模型配置列表（LLM和Embedding模型）
- [ ] 系统支持新增模型配置（名称、类型、API地址、密钥、模型名称）

- [ ] 系统支持删除模型配置（删除启用中模型时提示先切换）
